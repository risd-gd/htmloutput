Initial HTML explorations. While getting familiar with code, I explored creating illustrations with div boxes. Here's my process of trying to create fly swatter.

As I continued to get more comfortable with the HTML, I started to develop more repeated pattern forms. With this collection of patterns, I began to consider where and how these could be apply to different surfaces. I created versions of my HTML prints for computer wallpapers and printed wrapping paper.
