While getting familiar with code again, I explored creating illustrations with div boxes. Here's a few HTML process sketches of trying to create a fly swatter and ticket stubs.

As I continued to get more comfortable with the HTML, I started to develop more repeated pattern forms. With this collection of patterns, I began to consider where and how these could be applied to different surfaces. I created versions of my HTML prints for computer wallpapers and printed wrapping paper.
