This website book combines conversations with practicing designers with three months of experimental projects that use the web browser to create off-browser forms: event ephemera, wallpaper and textiles, videos and a book. This book in your very hands was printed from a web browser. This website prints to a small paperback book. A "responsive" website that at any point in its development couldoutput to a pdf.

The two forms each offer something distinct, and complementary. The web version is search-indexible, and widely available. A book is an ideal place for designers to archive and present their work in a physical format. Unlike the web, the book is a final format. It cannot lose compatibility or functionality. It cannot be linked to, it cannot be updated, and it cannot be edited. It can be reprinted but each individual book is an archive of a specific time and state. This one, for example, was generated at XX/XX/XXXX at XX:XX .

Both website and book come from the same codebase. Shared with you from a Github repository.
